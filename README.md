## Cine Stream
A modern movie streaming web application that allows users to explore, search, and watch movies and TV shows with a clean and responsive interface.

---
## Features
- Search Movies & Shows
  Find your favorite movies and TV shows instantly.
- Watch Content
  Stream movies using embedded video players.
- Browse Categories
  Explore content by genres like Action, Comedy, Drama, etc.
- Movie Details Page
  View ratings, release dates, descriptions, and posters.
- Responsive Design
  Works smoothly on desktop, tablet, and mobile devices.
---
## Tech Stack
- Frontend: React.js
- Styling: Tailwind CSS / CSS
- API: Movie Database API (TMDB or similar)
- HTTP Client: Fetch / Axios
